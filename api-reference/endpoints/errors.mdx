## ErrorResponse

```json
{
  "code": "validation_error",
  "message": "One or more validation errors occurred.",
  "details": {
    "field": "amountMinor",
    "issue": "required"
  }
}
```

Fields:
- code: Machine-readable code
- message: Human-readable summary
- details: Optional structured context
